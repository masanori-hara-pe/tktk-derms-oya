Blobに保存されているCSVファイルの参照デモ

CSVファイルのリストを開く。 リストからサンプルのテストファイルを選択すると当該ファイルがダウンロードされます。

Blobの静的Webサイトホスティングの利用

本サイトは、Azure Blobストレージの静的Webサイトのホスティングサービスを利用しています。 静的サイトとは、HTML、CSS、JavaScriptで構成された、事前に用意した一定数のファイルを用いてページを配信するウェブサイトで、バックエンドのサーバー側の処理やデータベースはありません。 Blobストレージの静的Webサイトは、最も簡単に低コストで静的サイトを公開する方法ですが、機能が限定されています。 ファイルを公開する場合にMicrosoftのIIS（Webサーバー）のようにディレクトリの内容が参照できないため、そこに保存されているファイルのリストを作る必要があります。このデモでは、このリストを何らかの方法でCSVファイル蓄積時にhtmlファイルとして生成するというストーリーになっています。 また、カスタムドメインによるアクセスはサポートしていない、認証ができないという欠点があります。これらの欠点を解決する別のサービスが提供されています。 静的 Web サイトのホスティングは無料で利用できます。 サイトで利用した BLOB ストレージと、運用コストに対してのみ課金されます。

静的Webサイトを生成するツール

静的サイトのコンテンツを作成するための便利なツールがあり、広く普及しています。このツールを一般的に静的サイトジェネレーターと称し、上記のデモコンテンツはLumeというツールを使用して生成しました。LumeはDenoで動作する静的サイトジェネレーター(SSG)です。

公式ドキュメントによると、以下のような特徴を持っています。

  • マークダウンやJavaScript、JSX、Nunjucksなど、複数フォーマットに対応
  • 各プロセッサーのフックして柔軟に拡張可能
  • Denoベースのランタイム環境
